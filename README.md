# 2026 世界盃淘汰賽預測器

這是單檔 GitHub Pages 版本。只需要上傳 `index.html` 到 GitHub repository 根目錄即可使用。

## 使用方式
1. 到 GitHub repository 首頁
2. 點 `index.html`
3. 點右上角鉛筆 Edit
4. 全選刪除舊內容
5. 貼上新版 `index.html` 內容，或刪除舊檔後重新上傳本檔
6. 到 Settings → Pages → Deploy from a branch → main / root

## 功能
- SVG 國旗
- 拖曳預測
- 自動儲存
- 複製分享網址
